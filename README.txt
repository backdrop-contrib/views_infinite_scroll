INTRODUCTION
------------

The Views Infinite Scroll module provides an infinite scrolling pager for Views. The pager can be triggered with the press of a button or automatically as the user scrolls to the bottom of the view's content.

* For a full description of the module visit https://www.drupal.org/project/views_infinite_scroll

* To submit bug reports and feature suggestions, or to track changes visit https://www.drupal.org/project/issues/views_infinite_scroll


REQUIREMENTS
------------

This module requires the following module:

* Views - https://drupal.org/project/views


INSTALLATION
------------

* Install the Infinite Scroll module as you would normally install a contributed Drupal module. Visit https://www.drupal.org/docs/7/extending-drupal-7/installing-contributed-modules-find-import-enable-configure-drupal-7 for further information.


CONFIGURATION
--------------

1. Navigate to Administration > Modules to enable the Views Infinite Scroll module, Views and the Views UI modules.
2. Navigate to Administration > Structure > Views and Add New View. 
3. In the "Page Settings" fieldset, be sure to enable "Use a pager" by checking the appropriate box. Select "Continue and edit".
4. Expand the Advanced settings, in the Advanced settings tab enable "Use AJAX: Yes". Save changes.
5. Under Page Settings, in the "Pager" field set click on the first link after "Use pager"that shows the current pager (default is "full").
6. Select "Infinite Scroll". Apply the changes.
7. In the field set "Infinite Scroll Options" enter the number of Items to Display. For no limit enter "0". Apply changes.
8. Save configuration.
